# 사전평가 벤치마크 기반 채점 보정 보고

작성일: 2026-05-22

## P — Plan

목표는 실제 사전평가 엑셀에서 반복되는 오차를 이용해 자동평가의 보정 우선순위를 정하는 것이다. 2025 사전평가와 2026 데이터셋은 기준 연도가 다르므로, 2026 배점을 2025 값으로 직접 덮어쓰지 않고 반복 후보를 별도 시트로 분리해 검토 가능하게 한다.

## D — Do

- 웹 벤치마크 파서가 실제 사전평가 엑셀의 `Question Number`, `D(점수)`, `D(배점)` 구조를 읽도록 개선했다.
- 벤치마크 비교 결과를 `점수 차이`와 `배점 차이`로 분리했다.
- 로컬 벤치마크 파서에 `Calibration Candidates` 시트를 추가했다.
- 여러 기업에서 반복되는 배점 불일치와 반복 감점 갭을 자동으로 모으도록 했다.

## C — Check

확인한 주요 패턴:

- 총 352개 문항 점수행을 NHN, OCI, 현대트랜시스 사전평가에서 추출했다.
- 배점 비교 결과는 194개 일치, 155개 불일치, 3개 데이터셋 미존재였다.
- 반복 감점 갭 후보는 `7.30.14`, `7.30.1`, `5.11.6`, `7.54.3`, `5.1.2` 등이 우선순위로 잡혔다.

## A — Act

다음 보정 방향:

- `Repeated denominator mismatch`는 2025→2026 방법론 변경 여부를 먼저 확인한다.
- `Repeated deduction gap`은 증빙 매핑과 답변 개선안 생성 로직 보정 후보로 우선 검토한다.
- 실제 2026 사전평가 샘플이 생기면 같은 파서로 다시 실행해 기준연도 차이와 자동평가 오차를 분리한다.
