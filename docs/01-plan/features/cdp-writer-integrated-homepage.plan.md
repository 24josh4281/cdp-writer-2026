## 1. Current Situation

- GitHub 저장소는 비어 있는 초기 상태다.
- 기존 로컬 프로젝트에는 2025 CDP Climate Change CH 섹터용 문항별 최대득점 모범답안 JSON이 있다.
- 기존 웹앱에는 평가 화면과 작성 도우미 기능이 분리되어 있었고, 이번 작업은 독립 저장소용 통합 홈페이지로 재구성한다.

## 2. Objective

1. 문항별 최대 득점 경로를 보여준다.
2. 지속가능경영보고서/증빙자료 내용을 넣으면 글자수에 맞춰 답변 초안을 생성한다.
3. 생성 답변이 D/A/M/L 기준을 어떻게 충족하는지 즉시 표시한다.
4. 감점/부분충족 요소만 따로 보여주고 개선 문장을 제안한다.
5. 향후 10개 이상 기업, 2026 평가기준, 산업별 특수문항으로 확장 가능한 구조로 만든다.

## 3. Initial Scope

- 정적 프론트엔드: `index.html`, `app.js`, `styles.css`
- 로컬 서버: `server.py`
- 데이터: `data/cdp_cc_ch_model_answers.json`
- 지원 파일 입력: DOCX, PPTX, XLSX, TXT, CSV, JSON 텍스트 추출
- PDF는 선택적 라이브러리가 있을 때만 추출하고, 없으면 수동 붙여넣기를 안내한다.

## 4. Risks And Assumptions

- 현재 데이터는 2025 CC CH 중심이다. 2026 평가방법론은 새 기준서가 확보되면 데이터 재생성이 필요하다.
- 자동 충족 판단은 보수적 신호 기반 QA다. 공식 CDP 점수 확정이 아니라 사전검토용이다.
- 증빙자료의 원문 품질이 낮거나 문항과 관련 없는 경우 초안 품질도 낮아진다.
- 브라우저만으로 Office 파일을 안정적으로 읽기 어렵기 때문에 로컬 Python 서버가 파일 추출을 담당한다.

## 5. Success Criteria

- 로컬에서 `python server.py` 실행 후 홈페이지 접속 가능
- 67개 정성 문항 로드
- 문항 선택 시 D/A/M/L 최대득점 기준 표시
- 증빙 텍스트 기반 답변 초안 생성
- 충족/부분충족/보완필요 판정 표시
- DOCX/PPTX/XLSX 파일 업로드 후 텍스트 추출 가능
- CSV/JSON 내보내기 가능
