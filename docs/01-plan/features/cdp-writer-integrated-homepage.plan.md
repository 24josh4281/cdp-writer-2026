# CDP Writer Integrated Homepage - Plan

## Executive Summary

| 관점 | 요약 |
| --- | --- |
| 문제 | CDP 평가방법론, 문항 응답, 증빙자료, 감점 후보가 여러 파일과 화면에 흩어져 있어 반복 기업 평가에 시간이 오래 걸립니다. |
| 해결 | 문항 탐색, 작성안내, 평가기준, 배점, 증빙 입력, 답변 초안, 감점 점검을 하나의 로컬 웹앱에서 처리합니다. |
| 사용자 효과 | ESG/CDP 컨설턴트가 문항별 최대 득점 경로와 보완 필요 요소를 빠르게 확인하고 답변 초안을 작성할 수 있습니다. |
| 핵심 가치 | 반복 평가 작업의 추적성, 속도, 품질을 높이고 향후 2026 기준 및 다양한 기업 응답으로 확장할 수 있습니다. |

## Context Anchor

| 항목 | 내용 |
| --- | --- |
| WHY | 빠른 작성보다 평가방법론과 실제 답변의 일치 여부를 확인하는 것이 중요합니다. |
| WHO | 비개발자 ESG/CDP 컨설턴트가 실무 프로젝트에서 사용합니다. |
| RISK | 자동 생성 답변을 공식 점수로 오해하거나 증빙 없는 문장을 제출하면 품질 리스크가 생깁니다. |
| SUCCESS | 문항별 작성안내, 평가기준, 배점, 증빙 요구사항, 감점 후보가 한 화면에서 확인됩니다. |
| SCOPE | 2026 CDP Full Corporate 데이터셋을 기본으로 하고, 기업별 응답 평가와 작성 보조 기능을 확장합니다. |

## Current Situation

- 2026 CDP 질문지 및 평가방법론 기반 데이터셋이 `data/cdp_2026_full_dataset.json`에 구축되어 있습니다.
- 웹앱은 `index.html`, `app.js`, `styles.css`, `server.py`로 구성된 로컬 실행 구조입니다.
- DOCX, PPTX, XLSX, PDF 등 증빙자료 텍스트 추출 기능이 포함되어 있습니다.
- 국문 표시 품질은 1차 개선되었고, 문항명 84개는 수동 교정 규칙으로 보완되었습니다.

## Objectives

1. M1~M13 문항을 국문 중심으로 탐색합니다.
2. 문항별 작성안내, 평가기준, 배점, 태그/변경사항을 확인합니다.
3. 증빙자료와 키워드를 기반으로 답변 초안을 생성합니다.
4. D/A/M/L 기준 충족 여부와 감점 후보를 확인합니다.
5. 작성 결과를 CSV/JSON으로 내보냅니다.
6. 작업 중 새로고침이 발생해도 브라우저 탭 임시저장으로 초안을 보호합니다.

## Risks And Assumptions

- 자동 평가는 공식 CDP 점수 확정이 아니라 사전평가 보조입니다.
- 기업별 산업 특수 문항과 필수조건은 원문 기준으로 최종 확인해야 합니다.
- 브라우저 임시저장은 탭 단위 편의 기능이므로, 공식 백업은 CSV/JSON 내보내기를 사용해야 합니다.
- 민감한 기업자료를 입력한 경우 임시저장 초기화가 필요하며, 외부 터널은 사용 후 즉시 종료해야 합니다.
- 파일 업로드 API는 허용 확장자, 파일 크기, 파일 수, 압축 해제 크기, PDF 페이지 수를 제한해야 합니다.

## Success Criteria

- `python server.py --port 8780` 실행 후 `http://127.0.0.1:8780` 접속 가능
- 2026 정성 문항 370개 로드
- 국문 화면에서 일반 영어 혼용 토큰 0건 유지
- 문항명 주요 품질 플래그 0건 유지
- 새로고침 후 초안, 증빙 입력, 선택 문항 복구
- CSV/JSON 내보내기 정상 동작
